## 2.1 NumPy

NumPy (Numerical Python) is a powerful Python library used for numerical computing and scientific calculations. It provides support for multi-dimensional arrays, matrices, mathematical functions, statistical operations, and efficient data manipulation. NumPy is widely used in Machine Learning, Data Science, Engineering, and Scientific Research due to its high performance and ease of use.

### Topics Covered

* Array Creation
* Indexing and Slicing
* Broadcasting
* Mathematical Operations
* Statistical Functions
* Random Number Generation
* Reshaping and Stacking Arrays
* Boolean Indexing and Conditional Selection

## 2.2 Pandas

Pandas is an open-source Python library designed for data manipulation and analysis. It provides powerful data structures such as Series and DataFrame, which allow users to efficiently organize, clean, transform, and analyze structured data. Pandas is commonly used in Exploratory Data Analysis (EDA) and Machine Learning workflows.

### Topics Covered

* Creating DataFrames and Series
* Reading and Writing CSV Files
* Data Inspection and Exploration
* Data Cleaning and Preprocessing
* Row and Column Operations
* Indexing using `loc` and `iloc`
* Sorting and Filtering Data
* GroupBy Operations
* Statistical Summaries

## 2.3 Matplotlib

Matplotlib is a comprehensive Python library used for data visualization. It enables users to create high-quality graphs, charts, and plots for data analysis and presentation. Matplotlib supports a wide variety of visualization techniques and provides extensive customization options.

### Topics Covered

* Line Plots
* Bar Charts
* Horizontal Bar Charts
* Scatter Plots
* Histograms
* Pie Charts
* Plot Customization
* Titles, Labels, and Legends
* Grid Configuration
* Subplots and Multiple Visualizations
